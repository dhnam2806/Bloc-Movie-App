## App Features.

- Sign In, Sign Up, Sign Out account
- Reset password
- Home Screen
- Trending movie
- Popular movie
- Upcomming movie
- Top rated movie
- Movies detail
- Search Functionality
- Select favourite movie
- Movies trailer and poster
- Movie cast
